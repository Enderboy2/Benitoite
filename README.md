# Benitoite

It is a light Markdown note taking app


## Run Locally

Clone the project

```bash
  git clone https://github.com/Enderboy2/Benitoite.git
```

Go to the project directory

```bash
  cd Benitoite
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Features

- Live previews
- Cross platform

## Tech Stack
- Vite
- React
- Electron
- Tailwind css
